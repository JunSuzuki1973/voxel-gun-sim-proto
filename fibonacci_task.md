# Coding Test: Fibonacci with Memoization

Implement a function `fib(n)` that returns the nth Fibonacci number (0-indexed) using recursion with memoization.

Requirements:
- Use Python 3.
- The function should handle n up to at least 1000 efficiently.
- Include a main guard that prints fib(10) and fib(50) as example.
- Write the solution to a file named `fib_solution.py` in the current directory.

Example output:
```
fib(10) = 55
fib(50) = 12586269025
```
